# ShadowProof Bridge Executive Acquisition Packet

## Positioning

ShadowProof Bridge is a pilot-ready proof-governance bridge from model-generated mathematical drafts to Lean-oriented validation workflows. Its differentiator is the ShadowHoTT/bilattice state layer: acceptance, refutation, theorem drift, ambiguity, and human review are represented as first-class states rather than collapsed into a single pass/fail flag.

- Repair and retrieval scaffolds.
- HTTP/ASGI/MCP surfaces for pilot integration.
- Security hardening for command override rejection, admin route scoping, path root checks, request-size checks, and schema validation.
